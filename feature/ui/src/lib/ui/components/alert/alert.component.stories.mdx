# Alert Component

The `AlertComponent` is a reusable component that displays an alert.

## Usage

Here's an example of how you can use the `AlertComponent`:

<Canvas>
  <Story name="Default">
    {{
      template: `
        <Alert [alertTitle]="'Using ShadCn in Angular'">
          <p>
            You can now add nice and sleek components to your UI in a cinch
          </p>
        </Alert>
      `,
    }}
  </Story>

</Canvas>

<Canvas>
  <Story name="Provided icon">
    {{
      template: `
        <Alert
          [alertTitle]="'Using ShadCn in Angular'"
          [alertIconName]="'lucideBell'"
        >
          <p>
            You can now add nice and sleek components to your UI in a cinch
          </p>
        </Alert>
      `,
    }}
  </Story>
</Canvas>

<Canvas>
  <Story name="Destructive">
